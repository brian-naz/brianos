# BrianOS

To get started:
<a href="https://brianos.vercel.app/">Demo</link>

### Setup:

Clone the repo:
```bash
git clone https://github.com/brian-naz/brianos.git
```
Go to the directory and simply type:
```bash
npm install
```
To run:
```bash
npm start
```
